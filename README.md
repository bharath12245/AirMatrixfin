# AirMatrix Skyhigh Booking

## Project Info

**AirMatrix Skyhigh Booking** is a premium flight booking application powered by AI. It features smart fare predictions, secure booking flows, and an AI assistant "AeroSense" to help plan trips.

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

Prerequisites: Node.js & npm installed.

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd airmatrix-skyhigh-booking-main

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Features

- **Smart Fare Predictions**: AI-powered price analysis.
- **Secure Booking**: Bank-grade encryption.
- **Delay Insights**: Historical data on flight delays.
- **AI Assistant**: Personalized trip planning with AeroSense.
